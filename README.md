A simple AWS Lambda with pytest, coverage and mockito 

How to install
```bash
pip install -r requirements/local.txt
```

How to run test
```bash
py.test
```

How to run test with coverage
```bash
py.test --cov=basicLambda tests
```

How to build zip
```bash
make
```
